# Stack Tecnológico

- **Entorno:** Google Antigravity / Gemini CLI (entorno colaborativo ágil, alta iteración).
- **Stack por defecto (si no se especifica otro) [HARD]:** Python 3.11+ con SQLite, Flask para micro-servicios (solo si indispensable). Principio KISS extremo: si la app puede ser una SPA sin backend, no añadir backend. Si puede ser HTML+CSS+Vanilla JS, no añadir frameworks.
- **Prohibiciones [HARD]:** Nada de "Bloatware" (frameworks como Django para menos de 5 endpoints, librerías de utilidades que aporten menos de 5 funciones, dependencias que dupliquen funcionalidad nativa).
